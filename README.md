# Práctica 6 - Servidor Web Nginx con Docker

## Descripción
Este proyecto implementa un servidor web Nginx ejecutándose dentro de un contenedor Docker utilizando Docker Compose.

## 🛠 Tecnologías utilizadas
- Docker
- Docker Compose
- Nginx
- HTML5 + CSS3

## ⚙️ Cómo ejecutar el proyecto

1. Clonar el repositorio:
   git clone https://github.com/TU-USUARIO/practica6-nginx.git

2. Entrar al directorio:
   cd practica6-nginx

3. Levantar el contenedor:
   docker compose up -d

4. Abrir en el navegador:
   http://localhost:8080
